# C-Unplugged
C-Unplugged is a lightweight, command-line interface (CLI) application designed for efficient music library management. Built entirely in C, this project focuses on low-level memory management and dynamic data handling to provide a seamless user experience for organizing songs and playlists.

Core Features:

Dynamic Library Management: Add, remove, and search for songs using optimized Singly Linked Lists.

Hierarchical Organization: Organize tracks into Albums and custom Playlists.

Data Persistence: Integrated File I/O handling to ensure your music library is saved and loaded across different sessions.

Session History: Features a command-logging system to track user actions and session data.

Modular Architecture: Structured with clean, modular code to separate business logic from the user interface.

Technical Implementation:
Language: C

Data Structures: Linked Lists (for dynamic track management) and Structs (for data modeling).

Algorithms: Search and sort functionalities for organizing track metadata.

Tools: GCC Compiler, Makefile for automated builds.
